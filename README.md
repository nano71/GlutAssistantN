#### 推荐环境：
Android SDK 26-35  
Gradle 8.11.1  
OpenJDK 17.0.8.1 2023-08-24  
Kotlin 2.0.21
Flutter 3.24.3  
Dart 3.5.3  

#### 编译项目:

```
flutter build apk --obfuscate --split-debug-info=xxx_Struggle --target-platform android-arm,android-arm64,android-x64 --split-per-abi
```

或者

```
// Windows使用
./bulid.bat  

// Linux使用
./bulid.sh 
```
